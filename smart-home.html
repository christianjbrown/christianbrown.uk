🏗️ How it works

This page is a static site built with Jekyll and hosted on GitHub Pages. When it loads, its JavaScript calls two APIs through the Fastly CDN, which caches the responses and forwards any misses to a pair of Google Cloud Functions. One function reads the indoor temperature and humidity from our Samsung SmartThings sensors; the other fetches the outdoor forecast from the Met Office.

The moving parts are open source:

  • cloud-function-smart-home-temps — the Google Cloud Function that serves the indoor climate data.
  • php-smartthings-api-lib — PHP client for the Samsung SmartThings API.
  • php-cloud-function-lib — shared helpers for running the PHP Cloud Functions.
  • php-api-client-lib — the underlying HTTP API client library the above build on.
